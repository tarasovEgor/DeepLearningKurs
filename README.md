# DeepLearningKurs

A repository with coursework for the **Deep Learning** course  
at **ITMO University Master's program**.

## Description

This repository contains practical assignments, experiments, and materials completed as part of the course.  
The main focus is on applying deep learning methods in practice: from basic neural networks to modern architectures.

## Contents

- lab and practical assignments;
- experiments with deep learning models;
- notebooks with model training and evaluation;
- helper scripts and materials.

## Technologies Used

- Python
- PyTorch
- NumPy
- Pandas
- Matplotlib
- Jupyter Notebook
